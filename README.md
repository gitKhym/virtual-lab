# Godot Dialogic Project

## Overview

This project was developed in **Godot Engine** and uses the **Dialogic plugin** to manage in-game dialogue and branching story events.  
The project is organized into multiple scenes, each with its own script and logic, connected via Godotâ€™s signal system.

---

## Running the Project

### 1. Install Godot

- Download and install **Godot Engine (version 4.x or later)** from the official website:  
  [https://godotengine.org/download](https://godotengine.org/download)

### 2. Clone or Download the Repository

```bash
git clone git@github.com:gitKhym/virtual-lab.git
```

### 3. Open the Project in Godot

1. Launch **Godot Engine**.
2. Click **"Import"**
3. Navigate to the folder where you downloaded or cloned this project.
4. Select the file named **`project.godot`**.
5. Click **"Import & Edit"**.
6. Once the project opens, press **F5** to start the game.
